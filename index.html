01
⚛ ¿Qué es un Átomo?
Concepto base para emulación correcta

En el contexto de AIOS GameStation, un átomo es el archivo de ROM de un videojuego — la imagen binaria exacta del cartucho o disco original almacenada en tu pendrive USB o computadora.

ℹ
Cada átomo es un archivo único que contiene el software completo del juego. Son archivos como .nes, .gba, .iso, .bin, etc. El nombre "átomo" refleja que son la unidad mínima e indivisible del juego.

🟣 SNES ROM.sfc .smcSuper Nintendo256 KB – 8 MB

03
🔌 Cargar Átomo desde USB
Método paso a paso — el más eficiente
1
Abre el panel USB
Haz clic en el botón en el menú superior. Si ya tienes dispositivos conectados verás un número verde.
2
Conecta tu pendrive
Haz clic en ＋ CONECTAR DISPOSITIVO USB. El navegador abrirá un selector de carpetas — navega hasta tu pendrive USB y selecciónalo. No necesitas instalar nada.
ℹ
También puedes usar 📁 ABRIR CARPETA LOCAL si tus átomos están en el disco duro.
3
Navega hasta tus átomos
El explorador mostrará todas las carpetas y archivos. Los archivos ROM aparecen marcados con 🎮 ROM en verde. Puedes entrar en subcarpetas haciendo clic.
4
Elige cómo jugar
Tienes dos opciones:
▶ PLAY directo
Haz clic en el botón ▶ PLAY del ROM. Selecciona la consola destino y empieza a jugar inmediatamente.
+ COLA → importar
Agrega varios ROMs a la cola con + COLA, selecciona la consola y haz clic en ✓ IMPORTAR SELECCIÓN para añadirlos todos a tu biblioteca.
5
El átomo queda guardado
Una vez importado, el binario del átomo se guarda automáticamente en el almacenamiento local del navegador (IndexedDB). La próxima vez que abras el sitio, el átomo estará disponible sin necesitar el USB. Verás el indicador ⚡ OK en la tarjeta del juego.

05
🎮 Elegir el Emulador Correcto
Guía de motores EmulatorJS para cada átomo
✓
AIOS GameStation usa EmulatorJS como motor de emulación. Cada consola tiene un "core" específico. Usar el core correcto es fundamental para que el átomo funcione.
🟥
NES / Famicom
core: nes
.nes .fds .unf
⭐⭐⭐⭐⭐ Excelente
El mejor soporte. Casi todos los juegos funcionan perfectamente. Usa este para todos los átomos de NES/Famicom.
🟣
SNES / Super Famicom
core: snes
.sfc .smc .fig .bs
⭐⭐⭐⭐⭐ Excelente
Soporte casi perfecto. Si un juego no carga con .smc prueba renombrarlo a .sfc.
🔵
Game Boy Advance
core: gba
.gba .agb .gb .gbc
⭐⭐⭐⭐⭐ Excelente
Uno de los mejores emulados. El core gba también puede correr juegos de GB y GBC si los seleccionas aquí.
🔴
Nintendo 64
core: n64
.n64 .z64 .v64
⭐⭐⭐⭐ Muy bueno
Rendimiento depende del juego y PC. Para mejor rendimiento usa átomos en formato .z64 (big-endian). Si va lento, cierra otras pestañas.
🎯
PlayStation 1
core: psx
.iso .bin .cue .pbp .chd
⭐⭐⭐⭐ Muy bueno
Para juegos multi-disco usa formato .chd (comprimido) o .cue+.bin. El archivo .cue apunta al .bin correcto.
🔷
Sega Genesis / MD
core: segaMD
.md .gen .bin .smd
⭐⭐⭐⭐⭐ Excelente
Soporte completo incluyendo 32X y Sega CD si tienes los BIOS correspondientes.
⬛
Nintendo DS
core: nds
.nds .dsi
⭐⭐⭐ Bueno
Requiere más RAM. Puede ir lento en PCs de gama baja. Recomendado para PCs con 8GB+ RAM.
👾
Arcade (MAME)
core: arcade
.zip
⭐⭐⭐ Variable
Compatibilidad varía por juego. El archivo ZIP debe tener el nombre exacto del ROM de MAME (ej: sf2.zip para Street Fighter 2).
06
💾 Save States: Guardar y Continuar tu Partida
Sistema completo de guardado en AIOS GameStation
✓
AIOS GameStation tiene dos sistemas de guardado que trabajan juntos: el guardado interno del emulador (SRAM) y el sistema de Save States de la plataforma.
💾
Save State de Plataforma
Usa el botón 💾 SAVE en la barra del emulador o en el botón de la tarjeta del juego. Guarda el estado exacto en ese momento — como congelar el tiempo.
① Mientras juegas → pulsa 💾 SAVE STATE
② Para continuar → pulsa 📂 LOAD STATE
③ También desde la tarjeta → botón 📂 LOAD
🎮
Guardado Interno del Juego
Usa el sistema de guardado del propio juego (menú del juego → Guardar). Este se almacena en la memoria SRAM del emulador y persiste en IndexedDB.
① En el juego → accede al menú de guardado
② Guarda normalmente como en la consola real
③ Al reabrir → se restaura automáticamente
📤
Para transferir saves a otro dispositivo: ve a 💾 SESIÓN → 📤 EXPORTAR .aios. Este archivo contiene todos tus save states y átomos. En el otro dispositivo usa 📥 IMPORTAR .aios.
⬇
Save state individual: En la tarjeta de cada juego hay un botón ⬇ para descargar ese save específico como archivo .aios_save. Puedes cargarlo en cualquier momento con el botón ⬆ CARGAR ARCHIVO DE SAVES en el panel de sesión.
07
✈ Sesión Offline Sin Internet
Cómo usar la plataforma completamente sin conexión
1
Primera vez: carga con internet
La primera vez que juegas, necesitas internet para descargar el core del emulador (EmulatorJS CDN). El Service Worker lo guarda en caché automáticamente.
2
Carga tus átomos desde USB
Importa tus ROMs usando el método USB o + ROM. Los binarios se guardan en IndexedDB — ya no necesitas el USB después de esto.
3
Exporta tu sesión completa
Ve a 💾 SESIÓN → 📤 EXPORTAR .aios. Guarda el archivo en tu pendrive. Este archivo contiene todo.
4
Offline total
A partir de este punto, puedes usar la plataforma completamente sin internet. Abre el HTML, importa el .aios si es otro dispositivo, y juega. La barra inferior muestra ✈ OFFLINE cuando no hay conexión.
⚠
Nota importante: Los juegos de PS1 y NDS tienen archivos de 200MB–512MB. El archivo .aios exportado puede ser grande. Usa un pendrive con suficiente espacio.
08
🔧 Solución de Problemas
Errores comunes y cómo resolverlos
❓ El juego carga pero la pantalla está negra
El core no coincide con el tipo de átomo. Ve a la consola, edita el core. Por ejemplo, un .gba en una consola con core nes no funcionará. Crea una nueva consola con el core correcto.
❓ El emulador dice "No se pudo cargar EmulatorJS"
Sin internet y sin carpeta emulatorjs/ local. Solución: conecta a internet una vez para que el Service Worker cachee los archivos, o descarga EmulatorJS desde GitHub y coloca la carpeta junto al HTML.
❓ El ROM muestra 📁 RE-CARGAR en la tarjeta
El binario del átomo no está guardado en IndexedDB (posiblemente fue añadido en una versión anterior). Haz clic en ▶ PLAY y vuelve a seleccionar el archivo ROM una vez. Después quedará guardado permanentemente con ⚡ OK.
❓ El juego va muy lento
Cierra otras pestañas del navegador. Los juegos de N64, PS1 y NDS son exigentes. En Chrome activa la aceleración por hardware: Configuración → Sistema → "Usar aceleración de hardware cuando esté disponible".
❓ El save state no funciona / no guarda el progreso
El Save State de plataforma (botón 💾) funciona mejor con el guardado interno del juego. Para máxima fiabilidad: usa el menú de guardado del propio juego además del save state de la plataforma. El save state captura el estado del emulador; el guardado interno del juego es independiente.
❓ El USB no aparece en el detector
El API WebUSB/WebHID requiere Chrome o Edge. Firefox tiene soporte limitado. Solución alternativa: usa el botón 📁 ABRIR CARPETA LOCAL en el panel USB, que funciona en todos los navegadores modernos.
❓ Perdí mi sesión al limpiar el navegador
Siempre exporta un archivo .aios antes de limpiar el navegador. Ve a 💾 SESIÓN → 📤 EXPORTAR .aios y guárdalo en tu pendrive. Es tu backup completo.
